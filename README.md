# Pokédex

Una Pokédex interactiva con estética pixel art de la Gen 1, construida con HTML, CSS y JavaScript vanilla consumiendo la [PokéAPI](https://pokeapi.co/).

## Características

- Visualización de los primeros 151 Pokémon con imagen, número, nombre y tipos
- Filtro por tipo desde el panel izquierdo
- Filtro alfabético + lista navegable en el panel derecho
- Buscador por nombre o número de Pokédex
- Modal con estadísticas base, altura y peso al hacer clic en una tarjeta
- Los tres filtros interactúan entre sí en tiempo real
- Diseño pixel art con fuente Press Start 2P
- Totalmente responsive con drawers deslizantes en móvil

## Tecnologías

- HTML5 / CSS3 / JavaScript ES6+
- [PokéAPI](https://pokeapi.co/) — datos de Pokémon
- [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) — fuente pixel art

## Uso

Abre `main.html` en el navegador. No requiere instalación ni servidor.
